# Earth Nuke — дизайн

Дата: 2026-07-02. Одобрено пользователем в диалоге.

## Что это

Браузерная игра-песочница в одном файле `earth-nuke.html`, открывается локально двойным кликом.
3D-модель Земли, которую можно крутить мышью и бомбить ядерными зарядами кликом.
На месте ударов остаются постоянные разрушения (выжженные кратеры).

## Решения

- **Стек:** three.js с CDN (unpkg, ES-модули через import map). Работает с file:// —
  unpkg отдаёт `Access-Control-Allow-Origin: *`.
- **Текстуры:** `three-globe@2.31.0/example/img/earth-blue-marble.jpg` (цвет) и
  `earth-topology.png` (bump). При недоступности сети — процедурный fallback планеты.
- **Управление:** свой drag-rotate (две вложенные группы: наклон X + вращение Y) с инерцией
  и автовращением при простое; zoom колесом. Клик без drag = удар.
- **Удар:** raycast в сферу → боеголовка прилетает по нормали → вспышка + point light,
  огненный шар, кольцо ударной волны по поверхности, грибовидное облако из спрайтов
  (огонь additive + дым normal), тряска камеры, синтезированный взрыв через WebAudio.
- **Тайлы как на картах (добавлено позже):** при приближении (дистанция до поверхности < 1.2R)
  поверх глобуса подгружаются спутниковые тайлы Esri World Imagery (Web Mercator, z3–z10,
  LOD от дистанции камеры). Тайлы — изогнутые сферические патчи чуть выше базовой сферы
  (радиус растёт с зумом, чтобы детальный слой был поверх грубого); отбор — по углу видимой
  области (raycast через углы экрана), кэш текстур с LRU ~400. Полюса (>85°) закрывает
  базовый глобус.
- **Постоянные разрушения:** кратеры — декали (сферические патчи с генерируемой canvas-текстурой)
  над тайлами, видны на любом зуме. Раскалённая кайма — additive-декаль, остывает ~10 с.
  (Изначально кратеры выжигались прямо в canvas-текстуру глобуса, но тайлы бы их закрывали.)
- **UI:** счётчик бомб и мегатонн, выбор мощности (1/10/100 Мт — влияет на масштаб взрыва
  и кратера), кнопка «Восстановить планету», подсказка по управлению.

## Вне скоупа

Миссии/цели, мультиплеер, звук из файлов, мобильный pinch-zoom.
