## Воксельная кора (2026-07-07)

- ✅ **Состояние коры** (`src/crust/`): cube-sphere (6 граней × 256×256×8 вокселей), 384 ленивых чанка
  (32×32×8), Uint8 материал-id; материалы по глубине (грунт/порода/базальт); landmask=вода исключена из
  carve; детерминизм (одна последовательность carve → одно состояние).
- ✅ **Carve API** (`Crust.ts`): `carve(dir, radiusRad, depthVox, seed)` выгрызает эллипсоид по текущей
  поверхности (повторные удары копают глубже); рваные края (seed); счётчик `removedVoxels` (метрика целостности `integrity()` — этап 3);
  `reset()`.
- ✅ **Мешер Surface Nets** (`surfaceNets.ts`, `chunkGeometry.ts`): гранёные куски без ступенек,
  1-воксельный margin для нормалей/стыков, дедуп квадов между чанками, крышка на r=1 прижата к сфере.
- ✅ **Гибрид-рендер** (`CrustView.ts`, `HoleMask.ts`, `MagmaCore.ts`): нетронутая планета — гладкая
  сфера (GlobeView); повреждённые регионы — воксельные чанки Surface Nets. Маска дырок (equirect,
  wrap/полюсы учтены) → GlobeView discard; воксельный меш рисует яму+нетронутый край. Цвет: биом +
  палитра слоёв + гарь из DamageField. Дно пробития — эмиссивная магма на r≈0.94 (пульсация).
- ✅ **Displacement-кратеры удалены** из `DamageField.ts` (R-чаша, нормали-микрорельеф, vertex
  displacement — демонтированы); поле остаётся ТОЛЬКО для перекраски (G-гарь, B-полынья); морфология
  воксельная.
- ✅ **Интеграция** (`Scene.startExplosion`): `crust.carve + damageField.splat` (G/B) для суши/льда;
  радиус/глубина по мощности (1Мт≈2-4, 10Мт≈7, 100Мт≈15 вокселей; глубина 1Мт=1-2, 10Мт=3, 100Мт=5
  слоёв); `planetReset` → `crust.reset()` + очистка маски.
- ✅ **Юнит-тесты** (vitest): детерминизм carve; углубление при повторном carve; монотонность объёма
  и integrity; oceан не карвится; Surface Nets на эталонном чанке (снапшот вершин/граней, замкнутость);
  пробитие всех слоёв.
- ✅ **Приёмка headless** (`scripts/accept/shots.mjs`, WebGL2): прогрессия 3×100Мт в точку (яма глубже,
  слои темнеют, магма на дне), скол на силуэте (лимб), reset чистит; консоль без ошибок компиляции.
- ⚠️ Отложено: wrap splat у антимеридиана (заявленное приближение); блочные берега из-за маски 512×256
  (не критично для воксельной коры).

  оставлены; HUD больше не обещает снимки.
- ✅ **Разрушаемость (GPU displacement)**: `render/DamageField.ts` — equirect поле урона
  (R=глубина/G=гарь/B=оплавление), splat воронки на детонацию, **ping-pong max** (WebGL2-совместимо,

- ✅ **Взрывы по материалу**: маршрутизация в `render/Scene.ts` по `surface`; вода —
  `render/WaterBurstView.ts` (столб/купол/кольцо, в поле НЕ пишет — следа нет); лёд — полынья.
  Общий построитель геометрии купола — `render/effects/geometryUtils.ts` (Explosion+WaterBurst).
- ✅ **Приёмка headless** (`scripts/accept/shots.mjs`, WebGL2 swiftshader): биом-глобус, воронка на
  суше, слияние двух воронок, водный столб без постоянного следа, круглая полынья на льду, reset
  очищает поле — консоль без ошибок компиляции шейдеров. Dev-хуки `__strike/__reset/__lookAt`
  живут в `src/debug/devHooks.ts` за `import.meta.env.DEV` (вырезаются из прод-бандла).
- ⚠️ Отложено/приближения: wrap splat у антимеридиана не сделан (заявленное приближение);
  low-res маска 512×256 даёт блочные берега; тон пыли гриба по биому и «пар» водного взрыва —
  не делали (YAGNI). Материал НЕ влияет на жертвы/геймплей (по решению — только визуал).
