## Чем занимаемся прямо сейчас

**Фича «Материал поверхности и разрушаемость» реализована** (ветка `feat/material-destructibility`,
12 задач по плану `docs/superpowers/plans/2026-07-05-material-destructibility.md`, спека
`docs/superpowers/specs/2026-07-05-material-destructibility-design.md`). Что сделано:
- **Материал в sim** (авторитетно): `sim/material.ts` (`materialAt` → суша/вода/лёд + биом по
  правилам широтных поясов+шум), `sim/landmask.ts` (реальные материки, запечённая маска 512×256 из
  Blue Marble, `scripts/gen-landmask.mjs`), `sim/geo.ts` (`dirToLonLat`). Событие `explosionStarted`
  несёт `surface`/`biome`. Геймплей (жертвы) НЕ изменён — материал пока только визуал.
- **Стилизованный биом-глобус** вместо спутника: `render/MaterialGlobe.ts` (процедурная биом-текстура),
  спутниковый слой Esri убран, слой подписей оставлен.
- **Разрушаемость (GPU displacement-поле)**: `render/DamageField.ts` — equirect поле урона
  (R=глубина/G=гарь/B=оплавление), splat воронки на детонацию, **ping-pong max** (WebGL2-совместимо),
  `cos(lat)`-коррекция у полюсов. Глобус вершинно вдавливается по полю + перекрашивается (гарь/полынья).
- **Взрывы по материалу**: суша — гриб+воронка; вода — `render/WaterBurstView.ts` (столб/купол/кольцо,
  без следа); лёд — полынья (тёмная вода + кайма льда). Общий `render/effects/geometryUtils.ts`.
- **Приёмка headless** пройдена (harness `scripts/accept/shots.mjs`): биом-глобус, воронка/слияние,
  водный столб без следа, круглая полынья, reset; консоль без ошибок шейдеров на WebGL2.

Следующий фокус — **финальное ревью всей ветки** и мёрж `feat/material-destructibility` (контроллер).

## Согласованные решения (из брейншторма, подтверждены реализацией)

- Сборка: **Vite + npm + TypeScript**. ✅ реализовано.
- Архитектура: **ECS (miniplex)**. ✅ реализовано.
- Рендерер: **WebGPU с откатом на WebGL2**, за интерфейсом `render/Renderer.ts`. ✅ реализовано,
  оба бэкенда проверены (обычный Chrome → webgpu, headless swiftshader → webgl2).
- Скоуп этапа 1: **порт с паритетом функций, без новых фич**. ✅ соблюдён — HUD/декали/звук
  из Task 10 — прямой порт демо, не новая функциональность.
- **Эффекты на TSL** (узловые материалы three), не сырой GLSL. ✅ везде (атмосфера, взрыв,
  частицы, декали).
- **Никакого динамического света** — устранённый источник лагов эталона. ✅ подтверждено
  стресс-тестом.
