# Проект «Ядерная стратегия» (рабочее название)

## ЯЗЫК ОБЩЕНИЯ — РУССКИЙ

**Всё общение с пользователем ведётся на русском языке.** Исключения — только технические
термины, названия библиотек, имена файлов/функций, код и команды (их не переводим).
Это правило действует всегда и во всех сессиях.

## Первое действие в новой сессии

Прочитай `memory-bank/` целиком — это «банк памяти» проекта. Он полностью описывает цель,
архитектуру и текущее состояние. Порядок чтения указан в `memory-bank/README.md`.

## Кратко о проекте

3D-Земля в браузере, по которой можно наносить ядерные удары. Сейчас это демо в одном файле
(`reference/earth-nuke.html`). **Конечная цель — многопользовательская онлайн-стратегия.**
Идёт перенос демо на полноценную архитектуру (Vite + TypeScript + ECS + WebGPU).
